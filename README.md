# eeekgamesbot
The eeek games bot

## To run 

1. Create a `config.json` and add `DISCORD_BOT_TOKEN`
    ```json
    {
      "DISCORD_BOT_TOKEN": "token goes here - ask connor",
      "clientId": "token goes here - ask connor",
      "RIOT_API_KEY": "token goes here - ask connor"
    }
    ```
2. Run `npm install`
3. Run `node index.js`
